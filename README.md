# Personal Website

## Architecture

Two deployments from one repo:

| Deployment | URL | How |
|---|---|---|
| Static site | `patrickcstephens.com` | GitHub Pages (root HTML files) |
| PatChat app | `chat.patrickcstephens.com` | Heroku (`app/`) |

Shared JS and CSS live in `resources/` and are served by GitHub Pages. The Flask app loads them directly from `https://patrickcstephens.com/resources/` so there is only one copy of each file.

---

## Before Pushing to GitHub

### If you edited any page content or layout

Static HTML files are **generated** — do not edit them directly. Edit the templates instead, then rebuild:

1. Edit the relevant file in `_templates/`:
   - **Page content** → `_templates/pages/<page>.html`
   - **Shared layout** (head, nav, sidebar, footer, scripts) → `_templates/base.html`

2. Run the build script from the project root:
   ```bash
   source venv/bin/activate
   python build.py
   ```

3. Commit both the template changes and the regenerated HTML files.

### If you added a new page

1. Create a new template in `_templates/pages/` (copy an existing one as a starting point).
2. Run `build.py` to generate the HTML file.
3. Add the new URL to `sitemap.xml` if it should be indexed.

### If you edited shared JS or CSS

Changes to `resources/js/` or `resources/styles/` take effect on both the static site and the PatChat app once deployed. No build step needed — just commit and push.

---

## Running PatChat Locally

```bash
cd app
source ../venv/bin/activate
python main.py
```

App runs at `http://127.0.0.1:5000`. Requires an internet connection — the Flask template loads shared JS/CSS from the live `patrickcstephens.com` site.

> **Note:** Nav and sidebar links will point to `http://127.0.0.1:5000/...` locally until the latest JS is deployed to GitHub Pages. This is expected.

## Repo Structure

```
├── _templates/
│   ├── base.html           # Shared HTML layout (edit this to change all pages at once)
│   └── pages/              # One template per page (edit these for page content)
│       ├── index.html
│       └── posts/
│           ├── index.html
│           ├── product-strategy.html
│           ├── assumption-validation.html
│           └── understanding-search-for-product-managers.html
├── build.py                # Renders _templates/pages/ → static HTML files
├── resources/
│   ├── js/                 # Shared JS (loaded by both static site and PatChat)
│   └── styles/             # Shared CSS
├── app/                    # Flask app (PatChat)
│   ├── main.py
│   ├── templates/index.html
│   └── static/
│       └── js/chat.js      # PatChat-specific JS only
├── index.html              # Generated by build.py — do not edit directly
└── posts/                  # Generated by build.py — do not edit directly
```
